# 游玩输入语义（Q1 / 0.5 + Q3 滚动）

> **权威说明：** 引擎行为在迁移完成前仍须与仓库文档一致；核心页面在所列主题上为权威说明。
>
> 自 `docs/play-input.md` 迁入并翻译。Host 命令表与绘制列表接线仍在仓库
> `docs/`。UI 节点树见 [MoonBit UI](./ui)。

Host 设备每帧映射为**意图码**；引擎在
`Engine::tick(intent, dt~, skip_held~)` 中应用。完整 Host 命令表见仓库
`docs/host-commands.md`；打包/导出接线见 `docs/draw-list-pack.md`；UI 树见
[MoonBit UI](./ui) / 仓库 `docs/ui-moonbit.md`。

## 意图表与代码

定义于 `render.intent_from_code` / `intent_to_code`，由
`host_web/js_glue/boot.js` → `export_frame(intent_code, dt_ms, skip_held)` 接线。

| 代码 | 意图 | 含义 |
|-----:|------|------|
| 0 | `None_` | 本帧无一次性输入 |
| 1 | `Advance` | 完成打字机、恢复裸 Yield，或激活聚焦按钮 |
| 2 | `SkipTyping` | 一次性完成打字/推进（兼容）；Host 优先用长按跳过 |
| 3 | `OpenMenu` | 游玩中且栈空 → `game_menu`；模态已开 → 弹出一层 |
| 4 | `ToggleAuto` | 切换自动推进（+ prefs `auto_mode`） |
| 5 | `MenuUp` | 上一个可聚焦项；**历史置顶**时 → 向上滚动一行 |
| 6 | `MenuDown` | 下一个可聚焦项；**历史置顶**时 → 向下滚动 |
| 7 | `MenuLeft` | 聚焦滑块减一档（或左向焦点） |
| 8 | `MenuRight` | 聚焦滑块加一档 |
| 9 | `OpenBacklog` | 游玩中且栈空 → 显示 `"backlog"` 模态 |
| 10+n | `Select(n)` | 提交选项索引 `n`（键 **1**–**9** → n = 0..8） |

**不是意图码：** `skip_held`（按住 **Control** 时每帧 `export_frame` 标志）；
`export_pointer` / `export_wheel`（独立 wasm 导出）。

## 默认按键（浏览器 Host）

| 输入 | 效果 |
|------|------|
| Enter / Space / Z | `Advance`（菜单上 = 激活聚焦按钮） |
| 点击（画布） | `export_pointer` 命中测试（见 [指针](#指针)）；非帧意图 |
| 滚轮（画布） | 在 ScrollView 上时 `export_wheel`（见 [滚轮](#滚轮)） |
| Esc | `OpenMenu`（打开 `game_menu` 或弹出模态） |
| ↑ / W | `MenuUp` |
| ↓ / S | `MenuDown`（纯 **S**；**Ctrl/Cmd+S** = 快速存档槽 0） |
| ← / → | `MenuLeft` / `MenuRight`（设置滑块） |
| A | `ToggleAuto` |
| H | `OpenBacklog`（Playing；带 Ctrl 组合时忽略） |
| 1–9 | `Select(0)`…`Select(8)` |
| **按住 Ctrl** | 每帧 `skip_held=1`（连发跳过） |
| Ctrl/Cmd+S / Ctrl/Cmd+L | 快速存/读 **槽 0**（非意图） |

冷启动为 **Title**（`title` 模态）。任一模态打开时叙事 Advance 冻结；仅栈顶
接收 UI 意图。

## 指针

引擎拥有命中测试（非 Host 布局常量）。Host 将画布事件映射到逻辑 1920×1080
坐标并调用 `export_pointer(x, y, phase)`：

| `phase` | 事件 | 引擎效果 |
|--------:|------|----------|
| 0 | `pointermove` | 更新悬停；内容/滚动条拖拽进行中则平移或映射滑块 |
| 1 | `pointerdown` | 命中激活、开始 ScrollView 拖拽，或 Playing 下未命中 → Advance |
| 2 | `pointerup` | 结束 ScrollView 拖拽；**绝不** Advance |
| 3 | `pointerleave` | 清除悬停并结束滚动拖拽 |

返回值 `hover_kind`：**0** 无，**1** 按钮，**2** 选项，**3** 滑块。
Host 映射为 CSS 光标（`pointer` / `ew-resize` / `default`）。

| 输入 | 效果 |
|------|------|
| 点击空白（Playing） | Advance（引擎；打字机先完成，与键盘相同） |
| 点击按钮 / 选项 / 滑块 | 命中激活 |
| 在 ScrollView 内容上按下 | 开始内容拖拽（指针 y 上移 → 更旧行） |
| 在滚动条拇指 / 轨道上按下 | 条拖拽；点轨道先跳转再继续 Bar 拖 |
| 移动 | 悬停 + 光标；活动拖拽更新 `scroll_y` |
| 抬起 | 仅结束拖拽 |
| 离开画布 | 清悬停 + 结束拖拽 |

**按下优先级**（模态 / Title）：可聚焦项 → 滚动条拇指 → 轨道 → 内容拖拽。
可聚焦项优先并清除任何拖拽。

**同帧规则：** 先 `export_pointer` / `export_wheel`，再
`export_frame(0, dt, skip_held)`。指针按下已消费交互 — Host 须将挂起意图置为
`None`（代码 **0**），避免同帧双重 `Advance`。指针 / 滚轮**不**推进
`wait_remaining` / 补间。当 `wait_remaining > 0` 时，按下不 Advance/Select。
Title / 模态栈上，空白点击不推进叙事。滚轮 / 滚动永不 Advance。

## 滚轮

Host 将画布 `wheel` 映射为逻辑坐标并调用 `export_wheel(x, y, dy)`：

| 项 | 规则 |
|----|------|
| 符号 | 引擎：**`dy > 0` 减小 `scroll_y`**（显示**更旧**行） |
| 浏览器映射 | Host 传 `dy = -event.deltaY`，使向上滚轮显示更旧 |
| 命中 | 模态已开 **且** `(x,y)` 在活动 ScrollView 视口内 |
| 外部 / 无模态 | 无操作；永不 Advance |
| 步进 | `scroll_y -= k * dy`，`k = 1`；钳制到 `[0, scroll_max]` |

打开 `"backlog"` 时在下一帧布局将滚动钉到最新内容（底部）。滚动状态在
`UiRuntime`（不入存档）。节点与主题角色见 [MoonBit UI](./ui)。

## `skip_held` 与 `SkipTyping`

| | `SkipTyping`（代码 2） | `skip_held`（tick 标志） |
|--|----------------------|-------------------------|
| 形态 | 一次性意图 | 按住 Ctrl 期间每帧布尔 |
| 打字机未完成 | 完成文本 | 完成文本（每次进入连发一次） |
| Yield 且文本已完成 | 类似 Advance 恢复 | 连发推进，每帧最多 **`skip_burst_max` = 8** 次 Yield→run |
| 定时 `@flow.wait` | **忽略**（门控） | **忽略**（门控） |
| 等待 `Choose` | 不自动选择 | **不**自动选择选项 |
| 模态 / Title 活动 | 仅 UI 路径 | 对叙事忽略该标志 |
| 遮罩淡入淡出进行中 | 不强制吸附完成 | 同 Advance（无额外 fade 门控） |

Host 将**按住 Ctrl** 映射为 `skip_held`，而非一次性 `SkipTyping`。自动模式在
空闲时合成 `Advance`；两者同时适用时，长按跳过更快。

**失焦 / 隐藏标签页清除粘滞跳过：** Host 在 window **blur** 与
`document.visibilityState === "hidden"` 时将 `ctrlHeld = false`。
`js_glue/boot.js` 与 `apps/host-web` 的 `gameSession.ts` 均实现。

## `wait_remaining` 门控

定时 `@flow.wait T`（`T > 0`）设置 `stage.wait_remaining = T` 并 yield。
当 `wait_remaining > 0` 时：

- **`Advance`**、**`SkipTyping`** 与 **`Select`** 被降为 `None_`。
- **`skip_held`** 不连发推进（wait > 0 时连发循环也会停止）。
- **`OpenMenu` / 菜单导航** 仍可用；模态打开时倒计时在后台继续。
- 裸 `@flow.wait` / `@flow.yield` 使 `wait_remaining = 0`，靠 Advance
  （或跳过连发）恢复。

图层补间、`trans.fade` 与 `trans.dissolve` **本身**不门控 Advance — 需要呈现
阻塞时，脚本须与 `@flow.wait` 配对。作者模式见 [MoonYuki 子集](./moon-yuki)。

### 呈现时钟 vs 等待 / 跳过

Dissolve 与 fade 时钟在定时 `@flow.wait` **期间继续推进**（菜单打开时亦然 —
同一呈现 tick）。长按跳过与 Advance 仍被 `wait_remaining > 0` 挡住，且**不会**
在中途 fade/dissolve 时吸附完成；等待结束后，剩余呈现时间继续按墙钟走到完成。

**中途 dissolve 存/读：** 双相 dissolve 不在存档格式中；读档后 dissolve 时钟被
硬清零（`dissolve_phase=0`）。作者不应依赖恢复进行中的 dissolve — 见仓库
`docs/host-commands.md`（trans / 存档格式）。

## 确认行为

危险 UI 动作走模态 `"confirm"` 与 `ConfirmKind`：

| 种类 | 触发 | Yes | No（默认焦点） |
|------|------|-----|----------------|
| `OverwriteSave(slot)` | 存入已占用槽 | 写入槽 | 取消 |
| `QuitToTitle` | 游戏菜单 → 标题（`request_quit_to_title`） | `quit_to_title` | 取消 |

- 文案为 `std_ui` 中固定的 `Literal("Are you sure?")`（不绑定 kind；
  `ConfirmKind` 只选择 Yes 动作）。
- **默认焦点为 No**，避免仅按 Enter 就确认破坏性操作。
- `confirm_yes` / `confirm_no` Capabilities 关闭确认流程。

## 历史记录（Backlog）行为

| 项 | 规则 |
|----|------|
| 来源 | 打字机完成后的完整对白行（`speaker` + 全文） |
| 容量 | 环形缓冲 **100**；丢弃最旧 |
| 持久化 | **仅会话** — 不入存档；在 `start_game`、`quit_to_title`、读档时清空 |
| 打开 | **H** / `OpenBacklog`，或游戏菜单 **History** → `"backlog"` |
| 关闭 | Esc / Close → `return_modal` |
| UI | 只读 `ScrollView`，最多 **100** 行 `BacklogLine(i)`；可自由滚动 |
| 打开钉位 | 打开模态时滚动钉到**最新**（底部） |
| 滚动输入 | 视口上滚轮；内容拖拽；滚动条；历史置顶时 **↑/↓** 按行步进（64 逻辑像素） |
| 焦点 | Close 为唯一可聚焦项；↑/↓ 始终滚动 |
| 不记录 | 选项标签、系统模态文案 |

历史页上的滚轮 / 拖拽永不 Advance 叙事（模态门控）。ScrollView 节点与主题角色见
[MoonBit UI](./ui)。
